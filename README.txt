Image.h:
This file declares all the classes and pmg image operations

Image.cpp:
This file defines all the special funcions and image and iterator methods

main.cpp:
Is the driver file where all operations are executed. You can type in after you make "./imageops.exe --h" to get help on how to structure your command line arguments. And if you enter it in incorrectly it also displays this menu.

ImageOpsUnitTests.cpp:
This file is where unit testing takes place. For the qualitative test I also test to check that the images are the correct dimension. You can execute all this tests using "make test" in the command line.
